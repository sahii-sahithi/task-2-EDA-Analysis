# 📊 Task 2 – EDA Analysis

Exploratory Data Analysis on Student Dataset using Python and SQL.

---

# 🎯 Objectives

The primary objectives of this project were:

- Perform Descriptive Statistics and Univariate Analysis
- Understand student demographic distribution
- Answer business questions using SQL queries
- Perform Multivariate Analysis and Correlation
- Create a static dashboard mock-up showing key metrics

---

# 📂 Dataset Overview

The dataset contains **1000 student records** with demographic and academic information.

| Column Name | Description |
|-------------|-------------|
| Student_ID | Unique identifier for each student |
| Name | Student name |
| Date_of_Birth | Student birth date |
| Gender | Gender of the student |
| Course | Enrolled course |
| Year | Academic year |
| City | Student city |
| Email | Student email address |

---

# 📊 Analysis Performed

### Descriptive Statistics & Univariate Analysis
- Gender Distribution
- Course Distribution
- City Distribution
- Academic Year Distribution

### SQL Business Questions
- Total number of students
- Gender distribution
- Course with highest enrollment
- Students per academic year
- Top cities by student enrollment

### Multivariate Analysis
- Age vs Academic Year scatter plot
- Course vs Gender distribution
- Correlation heatmap
- Pair plot analysis

# 📊 Dashboard Mock-up

A static dashboard was created to visualize key student metrics.

### Key KPIs

- Total Students
- Average Age
- Most Popular Course
- Top City by Enrollment
- Gender Ratio
- Students per Academic Year
